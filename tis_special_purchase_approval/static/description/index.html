Special Purchase Approval

Goto Purchase -> Product -> Purchase -> Enable the option "Purchase Manager Approval Required?"

If any product in the purchase order requires "Purchase Manager Approval" these purchase orders can only confirmed by the Purchase Manager.

If any User who is not a Purchase Manager tries to confirm any Purchase Order which contains any product which requires "Purchase Manager Approval" it will show a warning.
