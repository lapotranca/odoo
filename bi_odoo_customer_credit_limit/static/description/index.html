Odoo Customer Credit Limit with Sales Rules Odoo Apps

This odoo apps module enhance sales and accounting features, first it setup customer setup limit in Odoo, second you can make new credit rules for sales team for customer based on customer previous record you can writing down policy for credit management to that account in Odoo. Through this module, the user/sales person/sales team can define rules for checking the credit limit of the customer which also not over due payment or credit but also product and categories set on credit limit rules as advance and based on that sale order will get confirmed, if Credit Limit rules reached it limits on customer form in Odoo sale and accounting app.

Features

Allow user to set Credit Limit Rule on Customer.

Allow to set credit rule for any customer.

Allow to Set Credit Limit on Product Category.

User can apply credit limit for selected product category in credit limit rule.

Allow to Set Credit Limit for Specific Products.

User can apply credit limit for selected products in credit limit rule.

Allow to Set Credit Limit on Receivable Amount of Customer or Due Dates.

Select credit limit rule to show warning on customer receivable amount or the due date till days for that customer.
