Repair Management for Cars - Odoo Module

All Types of Car Repairs

This app is developed to handle business/industries of Car Repairs using Odoo.

Main Features :

  • Your customer can send a car repair request from your website and also attach documents while submitting a request from the website.
  • Generation of unique car repair request on submission and record it as repair request ticket in backend.
  • Customers can check the status of all car repair request tickets submitted by him/her on My Account page at Portal side.
  • Allow Repair manager / Repair users to create and manage diagnosys of car repair requests and create quotation and send them to your customer.
  • Allow Repair Manager / Repair User to create Job Order / Work Order from Repair Ticket and assigned to Responsible or Technician or Repair User.
  • Allow you to configure and manage Car Repair Teams, Services, Service types.
  • Allow you to configure and manage Cars and Cars Parts.
  • Allow you to add more images on the Car Repair Request ticket form.
  • Car Repair Request report in PDF format.
  • Car Repair User / Manager / Technician can communicate with customers using chatter and fill timesheets for work they have done for that ticket.
  • Car Repair Request Manager can close tickets and send bills to customers (Billing could be done from Quotation created from Diagnosys). Feedback requests will be sent to customers when the manager closes the ticket.
  • Customers can give feedback and rating of Car Repair Request by clicking on button.
  • Manage your Car Repair Request tickets using assignment to Car Repair Team.
  • Reports -> Car Repair Request in PDF, Car Repair Label and Car Repair Analysis Available.
  • FFor more details you can watch the video and see the screenshots below.

Roles by Users:
    1. Customer
    2. Repair Manager
    3. Repair User/Technician
    4. Sales User

    Role of Customer

  • Customers can create car Repair/Service requests from website.
  • Customers can view submitted Requests in My Account Portal.

    • Role of Repair Manager

  • Manager can Assign Requests to Repair User/Technician.
  • Manager can create Car Diagnosys from the ticket.
  • Print Pdf Report of car repair request.
  • Manager can close the ticket Request at the end of the car repair.
  • Managers can also create quotations from diagnosys.
  • Filling timesheets.

    • Role of Repair User/Technician

  • Repair User can see and manage Repair Request assigned by Repair Manager.
  • Repair Users can Create Work Orders and Work on Work Order / Job Order.
  • Repair Users can also Create and manage car Diagnosys from repair requests.
  • Repair User / Technician can communicate with Customers.
  • Filling timesheets.

    • Role By Sales User

  • Sales User can Send Quotation by Email to customer and work on quotation to make it order and then create invoice from order. (Odoo Standard process).
